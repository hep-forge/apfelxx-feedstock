[![Anaconda Platforms](https://anaconda.org/hep-forge/apfelxx/badges/platforms.svg)](https://anaconda.org/hep-forge/apfelxx)

Feedstock for [apfelxx](https://github.com/vbertone/apfelxx) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

A PDF evolution library in C++

## Install

```bash
conda install -c hep-forge -c conda-forge apfelxx
```
